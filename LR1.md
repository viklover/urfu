## Промпт для Claude Code

Проект на Java уже создан.  
Необходимо реализовать **10 отдельных Java-классов** (по одному на задание) в одном пакете (например, `lr1`).

В каждом классе должен быть:
- `public static void main(String[] args)`
- ввод данных через `Scanner`
- вывод данных в консоль

### Задания

Реализовать следующие программы:

1. Пользователь вводит фамилию, имя и отчество → вывести `Hello <ФИО>`.
2. Пользователь вводит имя и возраст → вывести информацию о пользователе.
3. Пользователь вводит день недели, месяц и число → вывести полную дату.
4. Пользователь вводит название месяца и количество дней → вывести сообщение о количестве дней в месяце.
5. По году рождения определить возраст пользователя.
6. Пользователь вводит имя и год рождения → вывести имя и возраст.
7. По возрасту определить год рождения.
8. Пользователь вводит два числа → вывести их сумму.
9. Пользователь вводит число → вывести:
   - число на 1 меньше,
   - введённое число,
   - число на 1 больше,
   - квадрат суммы этих трёх чисел.
10. Пользователь вводит два числа → вывести сумму и разность.

Код должен быть максимально простым, без лишних абстракций.  
**Один класс — одно задание.**

> Если встречаются задания, которые требуют решения задач с сайта **Timus (Тимус)** — **их необходимо игнорировать**.

## Отчёт по лабораторной работе

Также необходимо подготовить **отчёт по лабораторной работе в LaTeX**.

- Шаблон нулевой лабораторной работы находится в папке `docs`.
- Отчёт должен быть оформлен **строго по той же структуре**, которая указана в методических материалах.

### Структура отчёта

1. Титульный лист  
2. Цель работы  
3. Описание задачи
5. Ссылка на GitHub-репозиторий  
6. Вывод
